# 玩家控制说明

## 🎮 控制方式

### 鼠标控制

| 操作         | 功能                 |
| ------------ | -------------------- |
| **左键点击** | 选择星体             |
| **右键拖动** | 旋转视角（自由查看） |
| **中键拖动** | 围绕选中星体旋转相机 |
| **滚轮**     | 缩放（前进/后退）    |

### 键盘控制

| 按键               | 功能                |
| ------------------ | ------------------- |
| **W**              | 向前移动            |
| **S**              | 向后移动            |
| **A**              | 向左移动            |
| **D**              | 向右移动            |
| **Q**              | 降低视角            |
| **E**              | 抬高视角            |
| **Shift + 移动键** | 加速移动（3倍速度） |

## 🎯 使用技巧

### 选择和观察星体

1. **选择星体**：
   - 左键点击任意星体即可选中
   - 选中后会在控制台显示星体名称

2. **围绕星体旋转**：
   - 先用左键选中目标星体
   - 按住中键拖动鼠标
   - 相机会以星体为中心进行轨道旋转
   - 自动保持与星体的距离

3. **自由视角**：
   - 按住右键拖动鼠标
   - 可以自由旋转视角
   - 不会改变相机位置

### 相机移动

1. **基础移动**：
   - WASD控制水平移动
   - Q/E控制上下移动
   - 移动方向基于当前视角

2. **快速移动**：
   - 按住Shift + WASD/Q/E
   - 移动速度提升至3倍
   - 适合快速浏览大范围星系

3. **精确缩放**：
   - 使用鼠标滚轮前后移动
   - 沿着视角方向移动
   - 适合接近或远离目标

## ⚙️ 可调参数

在蓝图或代码中可以调整以下参数：

| 参数                  | 默认值 | 说明                |
| --------------------- | ------ | ------------------- |
| Camera Move Speed     | 1000   | 移动速度（单位/秒） |
| Camera Rotation Speed | 50     | 右键旋转灵敏度      |
| Camera Zoom Speed     | 500    | 滚轮缩放速度        |
| Fast Move Multiplier  | 3.0    | Shift加速倍率       |
| Orbit Rotation Speed  | 100    | 中键轨道旋转速度    |

## 🎬 操作示例

### 场景1：观察地球

```
1. 左键点击地球（选中）
2. 按住中键，拖动鼠标左右
   → 围绕地球水平旋转
3. 按住中键，拖动鼠标上下
   → 改变观察角度（俯仰）
4. 滚轮向前
   → 靠近地球
```

### 场景2：快速浏览星系

```
1. 按住Shift + W
   → 快速向前飞行
2. 按住右键，移动鼠标
   → 在飞行中调整方向
3. 松开Shift，继续W
   → 慢速接近目标
```

### 场景3：多角度观察恒星

```
1. 左键点击太阳（选中）
2. 按住中键，画圆圈
   → 360度环绕太阳
3. 滚轮调整距离
   → 找到最佳观察距离
4. 右键微调视角
   → 调整最终构图
```

## 🔧 高级技巧

### 轨道相机模式

当选中星体并使用中键旋转时：
- 相机始终面向选中的星体
- 保持与星体的距离不变
- 适合观察星体自转和轨道运动
- 可以用滚轮调整观察距离

### 自由飞行模式

不选中任何星体时：
- 完全自由的6自由度相机
- WASD/QE控制移动
- 右键控制视角
- 适合探索整个星系

### 组合操作

```
右键（视角） + W键（前进） + Shift（加速）
→ 快速沿视线方向飞行

中键（轨道） + 滚轮（缩放）
→ 围绕目标调整距离

Q/E（升降） + 右键（旋转）
→ 调整俯视/仰视角度
```

## 📝 注意事项

1. **鼠标光标**：
   - 鼠标光标始终可见
   - 方便点击UI和选择星体

2. **输入模式**：
   - 使用GameAndUI模式
   - 可以同时操作3D场景和UI

3. **性能优化**：
   - 相机移动使用DeltaTime
   - 确保不同帧率下速度一致

4. **碰撞检测**：
   - 当前相机可以穿透星体
   - 如需防止穿透，可在Pawn中添加碰撞
